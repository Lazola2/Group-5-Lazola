Pseudo classes and Pseudo selectors

Pseudo classes

A pseudo class is used to define a special state of an element.
It can be used to style elements when certain events happen on them like:

  • When a user hovers over an element
  • Styling visited and unvisited links
  • Style an event when it gets clicked
Examples include: hover, active, first-child, last-child, nth-child etc

Syntax:
selector:pseudo-class {
  property: value
}

Pseudo selectors

A pseudo element is used to style specified parts of an element.
It can be used to:

  • Style the first letter or a line of an element.
  • Insert content before, or after the content of an element
Examples include: first-letter, before, selection etc

Syntax:
selector::pseudo-selector {
  property: value
}
